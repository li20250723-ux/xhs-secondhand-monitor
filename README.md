小红书二手电视实时监控工具

本项目是一个基于 Python 和 DrissionPage 的自动化工具，用于实时监控小红书上的二手物品信息。

功能特点
- 锁定最新排序：强制切换小红书搜索结果为“最新发布”且检索前10个，记录过的不会再重复记录。
- 组合逻辑判定：同时匹配“目标词（电视等）”和“动作词（出、闲置等）”，精准筛选。
- 自动提醒：发现符合条件的帖子时发送系统桌面通知。
- 本地存档：自动将链接和日期记录到记事本。

使用方法
1. 安装依赖：`pip install -r requirements.txt`
2. 运行脚本：`python main.py`
3. 首次运行需在弹出的浏览器中手动扫码登录。
